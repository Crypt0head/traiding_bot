***TEST TRAIDING BOT***

This is a simple trading bot for general purpose

***Get Python dependencies***

**For Linux**

    sudo chmod +x setup_linux.sh
    ./setup_linux.sh

**For Windows**

    .\setup_windows.bat

***HOW TO RUN***

Simply run main.py from *root* (traiding_bot) dir

**For Linux**

    python3 test_bot/src/main.py

**For Windows**

    python .\test_bot\src\main.py
